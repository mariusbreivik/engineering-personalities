# Code Of Conduct

## Our Standard

This project aims to be critical about ideas and respectful toward people.

We welcome participation from anyone who wants to improve the repository in good faith. Disagreement is normal here. Hostility is not.

## Expected Behavior

- Be respectful and direct.
- Critique proposals, code, and wording without attacking people.
- Assume good intent, then ask for clarification when needed.
- Accept feedback without turning every thread into constitutional law.

## Unacceptable Behavior

- Harassment, discrimination, or intimidation.
- Personal attacks, insults, or mocking contributors.
- Deliberately inflammatory behavior.
- Publishing private information without consent.
- Using the project's skeptical tone as cover for being rude.

## Enforcement

Maintainers may remove, edit, or reject contributions and comments that violate this code of conduct.

Repeated or severe violations may result in temporary or permanent exclusion from project spaces.

## Reporting

Open a private maintainer contact channel appropriate for the hosting platform, or open an issue only if the report does not expose someone unnecessarily.

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and other project-managed spaces.
