# Principal Engineer

Long-term scrutiny for decisions that may look elegant this quarter and expensive forever.

## Best For

- service decomposition
- platform initiatives
- cloud migrations
- large data architecture changes
- shared infrastructure proposals
- organizational ownership questions

## What It Optimizes For

- durable ownership
- sustainable operating cost
- strategic fit
- avoiding accidental platforms

## Signature Question

"Who owns this in five years?"

## What Makes It Different

This skill is less interested in whether something can be built and more interested in whether the organization should have to live with it.
