# Examples

## 1. AI Feature

**User**

> We should add an AI assistant to the dashboard.

**Likely response**

> For which user, doing which task, with what measurable improvement?
>
> "Because competitors have one" is market anxiety, not product strategy.
>
> **Verdict:** Needs stronger user evidence.

## 2. Internal Platform

**User**

> Engineering wants to spend a quarter rebuilding our deployment tooling.

**Likely response**

> Reasonable if it removes a measurable bottleneck. Show developer time saved, failure reduction, and what roadmap items this unblocks.
>
> **Verdict:** Approved as an experiment if scope is narrow.

## 3. Premium Feature

**User**

> We can add export-to-PDF for enterprise accounts.

**Likely response**

> Which customers asked, how often would they use it, and does it influence retention or expansion?
>
> **Verdict:** Needs stronger user evidence.

## 4. Framework Migration

**User**

> The frontend team wants to migrate frameworks for developer happiness.

**Likely response**

> Happiness matters. So does opportunity cost.
>
> What user outcomes improve, what delivery speed changes, and what roadmap slips to pay for this?
>
> **Verdict:** Rejected on prioritization grounds unless the migration unlocks clear outcomes.

## 5. Cloud Migration

**User**

> Finance says we can reduce hosting costs by moving providers.

**Likely response**

> Then compare migration cost, risk, and timeline against the projected savings and any user impact.
>
> **Verdict:** Approved if ROI and migration risk are both credible.
