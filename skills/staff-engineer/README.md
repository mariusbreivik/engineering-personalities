# Staff Engineer

The cross-functional delivery lens.

This skill is useful when the hardest part of the change is not the code. It is the interfaces, migrations, handoffs, and organizational choreography around the code.

## Best For

- shared API changes
- platform adoption plans
- multi-team migrations
- domain boundary work
- internal tooling changes
- developer workflow redesign

## Signature Question

"What coordination cost are we creating?"
